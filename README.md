## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/thisisanimesh01/portfolio.git
cd portfolio
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
